# imegeri

Imageri lets you browse and manage your photos easily

## Platform

Android only

## Build

* Clone the project

    ```bash
    git clone https://github.com/ksb2311/imageri.git
    ```

* Go to folder
  
    ```bash
    cd imageri
    ```

* Run the project

  ```bash
  flutter run
  ```
